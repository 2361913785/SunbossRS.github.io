我们上次说了怎么在ubuntu18.04的系统下安装`unity`桌面环境，这次我们来讲如何下载`xfce4会话`
打开终端，输入命令：

```css
sudo apt install xfce4
```
这时，终端让你确认yes or no，输入y。  
  
然后终端就在下载xfce4桌面环境了。  
  
下载好后，重启。  
  
然后进入你的账户，在出现的“小齿轮”图标选择`xfce4`.
