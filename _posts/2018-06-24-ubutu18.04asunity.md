打开终端，输入命令：

```bash
#unity
sudo apt install unity-session

#xfce4
sudo apt install xfce4

#Cinnamon
sudo apt install cinnamon
```
这时，终端让你确认yes or no，输入y。  
  
然后终端就在下载unity桌面环境了。  
  
下载好后，重启。  
  
然后进入你的账户，在出现的“小齿轮”图标选择你的桌面环境。.
